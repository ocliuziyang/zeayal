> Description

Install

	$ git clone --depth=1 https://github.com/ocliuziyang/zeayal.git
	
    $ composer install
    
    $ yarn (or use $ npm install)
    
    $ gulp
    
    $ yarn run dev (or use $npm run dev)
    
    $ cp .env.example .env
    
    $ php artisan key:generate
    
    $ php artisan serve (also can use valet or homestead)
